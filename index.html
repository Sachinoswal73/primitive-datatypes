Primitive Datatypes - String, Number, Boolean, Undefined, Null

1. String

Any Character, Number, Special Characters or any Combination of these, which is assigned in " " or ' '.

Examples -
var fname; // a Property is Declared (not assigned / initialized yet ), The Datatype of this property is String. --- Output -->
var fname = "Sachin"; // characters --- Output :
var lname = 'Oswal'; // characters --- Output :
var phone = '902881939'; // number in the form of String --- Output :
var marks = "73.89"; // any type of number Int / Float but in the form of string --- Output :
var address = "B4-201, SPN, Pune." // Combination of characters, number & special characters in form of string --- Output :

2. Numbers

Any Number ( Integer, Float etc, mathematical calculation etc ) which is assigned without any '' or ""

<>Examples -
var spo = 73; // number --- Output :
var spo = 73.89; // number (float) --- Output :
var spo = spo + 7; // mathematical operations --- Output :

3. Boolean

Boolean can only be having 2 value. Either True or False, this value also should not assign in '' or ""

Examples -
var isBlack = True; // Boolean - True
var isBlack = False; // Boolean - False
--- Output :

4. Undefined

Properties which is defined but not assigned with any value, so any property defined with var has default value of undefined & type is also undefined.

Examples -
var ud; // value is undefined & datatype is also undefined --- Output :

5. Null

null == undefined - true but null === undefined - false.

Examples ---
var ud; // it's undefined now --- Output : undefined - undefined.
var udn = null; // null now --- Output :
